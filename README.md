# Lionsgate Real Estate Portal 

## Project Overview

The Lionsgate Real Estate Portal is a PHPMyAdmin and MySQL web application designed to manage real estate property listings.
The system allows different types of users to register, log in, browse properties, add listings, submit inquires, and manage real estate transactions. 

The platform supports different user roles: 
 - Agent
 - Buyer
 - Renter

Agents have the option to add and delete properites, while buyers and renters can view the listings and submit inquires.
